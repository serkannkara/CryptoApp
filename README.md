# CryptoApp

Hello! This is my new project CryptoApp. Project has made with Reactive Programming(Rxswift). Project shows the all crypto currency. Project shows the new crypto curency news.

## Features

- Programatically UI
- Rxswift
- Alamofire
- Kingfisher
- Charts
- Pagination

```
pod init

pod 'RxSwift', '6.5.0'
pod 'RxCocoa', '6.5.0'
pod 'Kingfisher'
pod 'Alamofire'
pod 'Charts'

pod install

```

Crypto API: https://www.coingecko.com/tr

News API: https://newsapi.org
